# Ram8-8-project-in-verilog
RAM8*8 design using verilog hdl
 the one 8 is no.of location
 The another 8 is width of each location(8 is 8 bits)
 wr.enb=1 is write in Ram
 wr.enb=0 is read in ram
 write address is 3 bit space
 read address is 3 bit space
 the entire design work on positive clk
 rst is Asynchoronus reset
 data in & out is 8 bit
